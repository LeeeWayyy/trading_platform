# Trading Glossary

Quick reference for trading platform terminology. See [GLOSSARY.md](../../../GETTING_STARTED/GLOSSARY.md) for the full glossary with all categories.

## Core Trading Concepts

- **Alpha** — Expected excess returns above a benchmark; model signal predicting returns (baseline IC=0.082)
- **Circuit Breaker** — Safety mechanism that stops trading when risk limits are violated (states: OPEN/TRIPPED)
- **Client Order ID** — Unique order identifier for idempotency: `hash(symbol + side + qty + price + strategy + date)[:24]`
- **Dry-Run** — Simulation mode (`DRY_RUN=true`): orders logged but not sent to broker
- **Idempotent** — Safe to repeat without side effects; critical for order submissions
- **Paper Trading** — Real APIs, simulated money (`DRY_RUN=false` + Alpaca paper credentials)
- **Position** — Current holdings of a security (symbol, quantity, avg price, unrealized P&L)
- **Reconciler** — Service syncing positions/orders between database and broker (boot-time + every 15 min)
- **Strategy** — Complete trading system: signals + allocation + execution (current: `alpha_baseline`)

## Orders & Execution

- **Fill** — Completed order execution at broker; updates positions and triggers P&L calculations
- **Market Order** — Immediate execution at best available price (no price guarantee)
- **TWAP** — Time-Weighted Average Price: splits large orders into smaller slices over time to minimize market impact
- **VWAP** — Volume-Weighted Average Price: `sum(price * volume) / sum(volume)`, benchmark for execution quality

## Risk & Safety

- **Drawdown** — Peak-to-trough decline in portfolio value (monitored continuously)
- **Kill Switch** — Emergency stop: cancels all orders, flattens positions, blocks new signals
- **Position Limits** — Maximum holdings per symbol or total portfolio (enforced pre-trade)

## Machine Learning & Signals

- **Alpha158** — Standard Qlib feature set with 158 technical indicators from OHLCV data
- **Backtest** — Strategy simulation using historical data (Qlib framework)
- **Feature Parity** — Research and production must share identical feature calculation code
- **Information Coefficient (IC)** — Correlation between predicted and actual returns (baseline: 0.082)
- **Model Registry** — Database storing ML model versions, metrics, promotion stages (dev -> staging -> prod)
- **Signal** — Trading recommendation from ML model (generated by Signal Service)

## Infrastructure

- **ADR** — Architecture Decision Record: documents important architectural decisions (`docs/ADRs/`)
- **Event Bus** — Redis pub/sub for inter-service communication (order fills, breaker state changes)
- **FastAPI** — Python web framework used for all microservices
- **Pydantic** — Data validation library using Python type hints (API schemas, configuration)
